# Proxivia API

This is the proxivia api.
For a wider documentation, please got to the /api-docs after launched the project

## Installation

Use github to clone or download the project, then install package dependencies:

```bash
npm i
yarn i
```

## Usage

```bash
npm run start
```

## Testing
```bash
npm run test
```
